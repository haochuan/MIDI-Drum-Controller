Documentation for Project 2
Haochuan Liu

1. To check the drum machine program, please run: "wxserpent64 drum_main.srp".

2. "drum_pattern.srp" is used to load and play drum pattern from a *.txt file.

3. There are four drum patterns in score*.txt. The language of the drum pattern is described as below. For example, this is from score1.txt:


beat  kick(vol)  snare(vol)  hihat(vol)
---------------------------------------
 1   |   100    |     0     |    100
 2   |    0     |     0     |     0
 3   |    0     |     0     |    100
 4   |    0     |     0     |     0
 5   |    0     |    100    |    100
 6   |    0     |     0     |     0
 7   |    0     |     0     |    100
 8   |    0     |     0     |     0
 9   |   100    |     0     |    100
 10  |    0     |     0     |     0
 11  |   100    |     0     |    100
 12  |    0     |     0     |     0
 13  |    0     |    100    |    100
 14  |    0     |     0     |     0
 15  |    0     |     0     |    100
 16  |    0     |     0     |     0

 Please note that each hihat is a sixteenth note.
